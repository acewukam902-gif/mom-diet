妈妈高性价比饮食计划

适用于51岁、157cm、约67.5kg、近期体重上升后的控重期。食材改为菜市场/普通超市常见款，避开三文鱼、蓝莓、藜麦、希腊酸奶等高价食材。

约1250–1400 kcal/天蛋白质约75–90g/天熟米饭90–120g/餐蔬菜≥500g/天一人一周版

1. 控重目标与低成本原则

27.4BMI约值，属于超重区间
1–2斤/月更稳妥的减重速度
75–90g每日蛋白质目标
18–28元每日食材粗估，按地区浮动

低成本不是少吃，而是把钱花在“高饱腹、高蛋白、高营养密度”的基础食材上：鸡蛋、豆腐、豆浆、牛奶、去皮鸡腿/鸡胸、草鱼/鲫鱼/鲭鱼、应季绿叶菜、白菜、圆白菜、胡萝卜、土豆、红薯、燕麦、小米、大米。

提示：如果1个月胖10斤伴随水肿、气短、乏力、尿量变化、明显怕冷便秘或异常出血，先做体检/就医排查，不要只靠节食处理。

2. 7天三餐不重样菜单

第1天：启动日约1320 kcal；蛋白质约82g；主食稳、蛋白足。
早餐约330 kcal

燕麦35g + 纯牛奶250ml + 水煮蛋1个 + 苹果半个。

午餐约520 kcal

米饭熟重110g + 去皮鸡腿肉120g + 清炒圆白菜250g + 胡萝卜50g。

晚餐约390 kcal

红薯150g + 白菜豆腐汤：北豆腐180g、白菜300g、香菇/平菇少量；凉拌黄瓜1根。

省钱替换：鸡腿肉可换鸡胸、鸡蛋2个或豆腐250g；香菇可不放。
第2天：家常控油日约1280 kcal；蛋白质约78g；适合普通家庭炒菜。
早餐约340 kcal

玉米1根中等大小 + 无糖豆浆300ml + 茶叶蛋1个 + 小番茄/番茄150g。

午餐约510 kcal

米饭熟重100g + 芹菜炒瘦猪肉90g + 豆芽炒胡萝卜250g，油合计不超过10g。

晚餐约380 kcal

小米南瓜粥1碗：小米30g、南瓜150g + 草鱼/鲫鱼块120g炖冬瓜250g。

省钱替换：鱼可用冷冻鲭鱼、带鱼段或鸡蛋豆腐羹替代。
第3天：豆腐鸡肉日约1340 kcal；蛋白质约86g；成本低、饱腹强。
早餐约350 kcal

小米南瓜粥1碗 + 水煮蛋1个 + 纯牛奶200ml。

午餐约520 kcal

米饭熟重110g + 鸡胸/去皮鸡腿肉100g拌黄瓜 + 家常豆腐150g炖白菜200g。

晚餐约400 kcal

青菜鸡蛋挂面：挂面干重50g、鸡蛋1个、青菜250g、豆腐丁80g；少油少盐。

省钱替换：挂面可换荞麦面或普通面条；重点是面量控制在干重50g左右。
第4天：鱼豆腐日约1300 kcal；蛋白质约80g；补钙友好。
早餐约330 kcal

全麦馒头/普通小馒头1个约60g + 鸡蛋1个 + 无糖豆浆250ml + 梨半个。

午餐约540 kcal

米饭熟重100g + 草鱼/鲫鱼150g炖豆腐120g + 清炒小白菜/上海青250g。

晚餐约360 kcal

土豆100g蒸/煮 + 鸡丝拌豆芽：鸡肉80g、豆芽250g、黄瓜半根；紫菜蛋花汤1碗。

省钱替换：鱼贵时用鸡蛋2个+豆腐200g补足蛋白。
第5天：番茄鸡蛋日约1270 kcal；蛋白质约76g；口味更家常。
早餐约330 kcal

燕麦30g煮豆浆250ml + 香蕉半根 + 鸡蛋1个。

午餐约500 kcal

米饭熟重100g + 番茄炒蛋：番茄250g、鸡蛋1个 + 瘦肉炒圆白菜：瘦肉70g、圆白菜250g。

晚餐约390 kcal

红薯150g + 青菜豆腐汤：豆腐200g、青菜250g；凉拌海带丝/木耳少量。

省钱替换：海带丝没有就换凉拌黄瓜；番茄贵时换冬瓜或白菜。
第6天：汤菜饱腹日约1360 kcal；蛋白质约84g；适合周末批量备菜。
早餐约350 kcal

蒸红薯120g + 纯牛奶250ml + 水煮蛋1个 + 黄瓜半根。

午餐约530 kcal

米饭熟重110g + 去皮鸡腿冬瓜汤：鸡腿肉120g、冬瓜300g + 清炒芹菜/青菜200g。

晚餐约410 kcal

鸡蛋豆腐蒸羹：鸡蛋1个、豆腐180g + 蒸南瓜150g + 蒜蓉油麦菜250g。

省钱替换：油麦菜可换任意当季绿叶菜；冬瓜可换白萝卜。
第7天：收尾稳定日约1290 kcal；蛋白质约79g；方便下周继续循环。
早餐约330 kcal

玉米面粥/玉米糊1碗 + 鸡蛋1个 + 无糖豆浆250ml + 橘子1个。

午餐约520 kcal

米饭熟重100g + 白萝卜炖瘦肉100g + 炒青菜250g + 凉拌胡萝卜丝少量。

晚餐约370 kcal

白菜鱼豆腐汤：普通鱼块120g、豆腐120g、大白菜300g + 土豆80g。

省钱替换：普通鱼块可换冻鱼、鸡胸、鸡腿肉或豆腐加鸡蛋。

3. 一人一周采购清单

主食

大米/糙米或杂粮米约700g；燕麦250g；小米150g；玉米3根；红薯/紫薯800g；普通挂面或荞麦面200g；馒头2个；南瓜500g；土豆500g。

蛋白质

鸡蛋10–12个；北豆腐4盒约1200g；无糖豆浆1–1.5L或黄豆300g；纯牛奶1.5–2L；去皮鸡腿/鸡胸700g；瘦猪肉300g；普通鱼类600–800g。

蔬菜

大白菜/圆白菜2–3kg；绿叶菜1.5kg；番茄1kg；黄瓜4根；胡萝卜5根；冬瓜/白萝卜1.5kg；豆芽500g；芹菜500g；洋葱2个；海带/木耳少量。

水果

苹果3个；梨2个；香蕉2–3根；橘子/橙子4个。每日水果约200g，不用买高价浆果。

调味

碘盐、低钠酱油、醋、蒜、姜、葱、胡椒、辣椒面少量；菜籽油/花生油/橄榄油任选，每日总油控制约20–25g。

可不买

三文鱼、鳕鱼、蓝莓、藜麦、希腊酸奶、坚果大礼包、代餐粉、酵素、减肥茶、蛋白棒。

4. 执行规则

如果饿了怎么加餐

优先选：牛奶200ml、无糖豆浆250ml、鸡蛋1个、苹果半个、黄瓜/番茄、豆腐汤半碗。不要用饼干、面包、坚果、奶茶解决饥饿。

如果家里人一起吃怎么做

全家菜一样做，给妈妈单独控制主食和油。她的盘子比例建议：半盘蔬菜，四分之一蛋白质，四分之一主食。

如果两周不掉秤

先查三件事：油是不是超量、熟米饭是不是超过120g、有没有零食饮料。确认执行到位后，再把晚餐主食减少30–50g熟重，或每天多走2000–3000步。

5. 风险提示

  • 如有糖尿病、慢性肾病、痛风、胆囊疾病、严重胃病、甲状腺疾病，菜单需要医生或营养师个体化调整。
  • 不要把热量压到1000 kcal以下；中年女性过度节食容易疲劳、暴食反弹和肌肉流失。
  • 近期快速增重可能与水肿、药物、睡眠、甲状腺、围绝经期激素变化有关；必要时做血糖、血脂、肝肾功能、甲状腺功能等检查。

6. 依据与说明

  • 中国居民膳食指南2022：食物多样、谷类为主；每天包括谷薯、蔬果、畜禽鱼蛋奶和豆类；平均每天12种以上食物、每周25种以上。
  • 中国居民膳食指南2022：餐餐有蔬菜，每天新鲜蔬菜不少于300g；每天水果200–350g；奶类相当于每天300ml以上液态奶。
  • 国家卫健委《成人肥胖食养指南（2024年版）》强调：成人肥胖患者膳食应全面、均衡、多样化，并限制总能量摄入。
  • 本方案按家庭可执行优先，热量和蛋白为估算值；不同品牌、烹调油量和食材含水量会造成差异。
